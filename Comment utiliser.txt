##### Lancer le logiciel CAnnovation #####

Prérequis :
Vous devez avoir installé sur votre machine :
 - Pyqt
 - Panda 1.8.1

Vous devez tout d'abord lancer :

 - socketbase_server2.py

Puis dans le dossier CAnnovation :

 - main.py

Puis entrer les coordonées de connexion, et cliquer sur "Ok"
